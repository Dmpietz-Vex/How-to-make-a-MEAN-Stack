How to Build a 'Hello World' Web App on MEAN Stack

1. When shell appears, you will need to install node.js, so type:curl -sL https://deb.nodesource.com/setup_12.x | sudo -E bash -

2. Type down: sudo apt install -y nodejs

3. To install express.js: Type command to make a directory: mkdir helloworld

4. Type command to change directory: cd helloworld

5. Type command to initialize your project which create a package.json: npm init

6. hit enter 3 times when you get to entry point type this:app.js

7. Hit enter 5 times until you get to the entry point again

8. Type command to run express.js and save it on your project and directory: npm install express --save

9. Type command: nano app.js

10. Copy the code below and paste it to the nano:

var express = require('express');
var app = express();
app.get('/', function (req, res) {
res.send('Hello there! Have a Great Day!!!');
});
app.listen(3000, function () {
console.log('Example app listening on port 3000!');
});

  • Press ctr+x to exit
  • It will ask if you want to save so type y then press keyboard ‘enter’

11. Go to console of Google Cloud Platform.Go to the top that shows search symbol,Go to the top that shows search symbol,There are two searches of ‘firewall rules,’ use one that is VPC network and click it

12. On the page of “Firewall rules,” go top to click CREATE FIREWALL RULE

13. For name, type ‘firewall-3000’ (3000 is a port that you will need to open on a google cloud platform. The one you had to write on nano showed the port 8080).

14. Change Targets to All instances in the network

15. On Source IP ranges, type: 0.0.0.0/0

16. On Protocols and ports, this will open port 3000: so click tcp and write: 3000

17. Click create

18. Go to the nav to find Compute Engine but don’t click and click VM Instances

19. Click the External IP of the instance’s name test-mean-stack which should show error and not working

20. Now go back to shellType the command: node app.js

  • It should show this:

21. Go to the External IP tab. Include http and 3000 to your numbers. http://00.00.000.000:3000/. You already have your External IP numbers so just use http:// and 3000/. Remember not to use ‘https’.

22. To leave the express.js, click the keyboard, ‘ctr+c’

23. Type the command to move up one directory: cd ..

24. To install MongoDB (the database), type down: sudo apt install -y mongodb

25. To check the status of MongoDB, type: sudo systemctl status mongodb

You will see what shows active: active running which means the database server is running and that you are connected to MongoDB.

26. Press Ctrl+c to get a command line.

27. To start MongoDB, type: sudo systemctl start mongodb

28. To work with database, type: mongo

30. Now we will install a web app.

31. Check if node is installed on the server machine on Google Cloud
node -v

32. Once you verified node is installed download get:sudo apt-get update

33. Install: sudo apt-get install git -y
