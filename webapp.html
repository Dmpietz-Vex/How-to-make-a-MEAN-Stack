How to build a web app on the MEAN Stack

(After you went through all the steps on the MEAN Stack tutorial...)

1. Now we will install a web app.

2. If your shell is closed, open up the shell click SSH

3. Check if node is installed on the server machine on Google Cloud : node -v

Git Installation on Ubuntu (4-6)

4. Once you verified node is installed download get:
sudo apt-get update

5. Install: sudo apt-get install git -y

6. To clone your web app github repository (ex: This tutorial uses Bonkai github repository):
git clone https://github.com/Dmpietz-Vex/Bonkai

7. To change “Bonkai” directory: cd Bonkai

8. To list files and directories: ls -la

9. To start Express.js: node routes.js

10. Once you get this message you're done.

11.Now go back to the google cloud platform and refresh your “hello world” page or follow the next steps. You should See the Bonkai web page.

12. Go to Hamburger icon -> Computer Engine -> VM Instances

13. Click the External IP

14. Add :3000 to the end of your server’s external IP

EX) http://34.105.103.160:3000 (do not copy and paste example, you have to use your external IP)
(remember not to use https)

15. Now your website is connected on the server

16. After #10, to get command line: ctrl c

17. To exit: exit
